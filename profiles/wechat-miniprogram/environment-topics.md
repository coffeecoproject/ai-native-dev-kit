# WeChat Mini Program Environment Topics

Use these topics when drafting `docs/environment-baseline.md` for a WeChat Mini Program project.

## Runtime

- appid ownership
- `project.config.json`
- WeChat Developer Tools version
- miniprogram root
- cloud function root
- local preview, upload, and build flow

## Cloud And Services

- CloudBase environment id
- cloud function deployment path
- database collection ownership
- storage bucket boundary
- access rules and permission ownership

## Permissions And Compliance

- privacy declaration
- user info, phone, location, camera, album, or notification permissions
- payment boundary
- content moderation or audit boundary

## Environment Variables

- cloud environment names
- function config names
- payment-related variable names
- third-party service variable names

Variable names may be recorded. Secret values must never be recorded.

## Release

- experience version flow
- upload/audit/production release owner
- rollback or hotfix process
- release evidence
- incident owner
