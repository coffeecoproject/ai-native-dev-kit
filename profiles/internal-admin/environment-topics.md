# Internal Admin Environment Topics

Use these topics when drafting `docs/environment-baseline.md` for an internal admin or management console.

## Runtime

- web frontend runtime
- backend/API dependency
- local dev and test commands
- role or seed data setup

## Governance

- RBAC or permission source
- audit trail
- sensitive operation approval
- data correction workflow
- import/export boundary
- batch operation boundary

## Environment Variables

- API base URL
- auth/session variables
- feature flag variables
- import/export service variables

Variable names may be recorded. Secret values must never be recorded.

## Release And Evidence

- admin preview/staging/production status
- release approver
- rollback owner
- audit evidence
- monitoring and alert owner
