# Android App Environment Topics

Use these topics when drafting `docs/environment-baseline.md` for an Android project.

## Runtime

- JDK version
- Gradle version
- Android Gradle Plugin version
- build variant or flavor
- emulator or device target

## Signing And Permissions

- signing config boundary
- keystore ownership
- manifest permissions
- push notification, location, camera, media, health, or payment boundaries

## Commands

- local build command
- local test command
- instrumentation test command
- bundle or APK command

## Environment And Release

- API environment switching
- internal testing, closed testing, or production release owner
- Play Console release process
- rollback, staged rollout, or hotfix status
- crash/log evidence

Secret values, keystore files, passwords, and production credentials must never be recorded.
