# Web App Industrial Bootstrap Kit

This bootstrap kit lists project-level assets a BL2 Web project should create or confirm.

Recommended project docs:

- `docs/baseline-selection.md`
- `docs/baseline-evidence.md`
- `docs/permission-model.md`
- `docs/verification-matrix.md`
- `docs/risk-policy.md`
- `docs/architecture.md`

Recommended evidence records:

- release record for production-impacting work
- incident record when an incident or recovery drill happens
- exception and residual risk entries for unmet industrial requirements

This kit is guidance only. It is not production readiness evidence.
