# Internal Admin Bootstrap Kit

Use this pack after the project has selected `internal-admin` and BL2 industrial mode.

Recommended first steps:

1. Audit roles, privileged operations, production data, audit trail, and rollback assumptions.
2. Draft project `docs/baseline-selection.md`.
3. Draft project `docs/baseline-evidence.md`.
4. Run `node scripts/check-industrial-baseline.mjs .`.
5. Use strict mode only after human approval.
