# CloudBase Runtime And Security Baseline

## Required Coverage

- environment separation and production configuration are documented
- cloud function input, permission, and side-effect boundaries are reviewed
- managed storage and database access rules are explicit
- secrets and platform configuration are reviewed before release
- deployment, rollback, and monitoring evidence exists
- exceptions and residual risks are recorded in project evidence

## AI Boundary

AI may prepare evidence and checks, but must not approve production environment changes, secrets, access rules, or residual risk.
