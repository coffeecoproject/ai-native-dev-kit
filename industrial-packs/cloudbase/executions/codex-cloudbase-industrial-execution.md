# Codex CloudBase Industrial Execution

## Before Implementation

- read `AGENTS.md`
- resolve selected profiles and industrial packs
- identify cloud function, access rule, storage, environment, deployment, and secret risk
- stop for human approval when the task touches production config, secrets, access rules, or value transfer

## During Implementation

- keep changes scoped to approved cloud capabilities
- update access rule, function boundary, data, deployment, and monitoring evidence
- avoid introducing implicit production dependencies
- record exceptions rather than silently weakening the baseline

## Before Final Report

- run available verification
- include function, access rule, environment, and deployment evidence
- list residual risks and approval scope
