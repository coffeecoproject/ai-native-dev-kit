# Payment And Value Transfer Bootstrap Kit

Use this pack after the project has selected BL2 industrial mode and value movement is in scope.

Recommended first steps:

1. Audit value movement, idempotency, reconciliation, audit trail, approvals, and mitigation assumptions.
2. Draft project `docs/baseline-selection.md`.
3. Draft project `docs/baseline-evidence.md`.
4. Run `node scripts/check-industrial-baseline.mjs .`.
5. Use strict mode only after human approval.
