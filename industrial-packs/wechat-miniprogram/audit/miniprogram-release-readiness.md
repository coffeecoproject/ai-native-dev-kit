# Mini Program Release Readiness Audit

## Audit Areas

- Experience version or review-like validation status.
- Platform category, permission declaration, privacy declaration, and review note changes.
- Release scope, rollback/mitigation path, monitoring path, and incident owner.
- Cloud function, access rule, production configuration, payment, and privacy impact.
- Human approval status for release, platform permission, value transfer, and residual risks.

## Output

Record:

- release readiness status
- blocking risks
- evidence refs
- required human decisions
- rollback or mitigation plan
- residual risk acceptance status
