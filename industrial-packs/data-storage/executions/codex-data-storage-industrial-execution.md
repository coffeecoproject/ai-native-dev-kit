# Codex Data Storage Industrial Execution

## Before Implementation

- read `AGENTS.md`
- resolve selected profiles and industrial packs
- identify schema, migration, deletion, backup, restore, retention, and regulated-data risk
- stop for human approval when the task touches production data, destructive behavior, or regulated data

## During Implementation

- keep changes scoped to approved data surfaces
- preserve compatibility unless approved otherwise
- update migration, integrity, recovery, and permission evidence
- record exceptions rather than silently weakening the baseline

## Before Final Report

- run available verification
- include migration, recovery, integrity, and data-boundary evidence
- list residual risks and approval scope
