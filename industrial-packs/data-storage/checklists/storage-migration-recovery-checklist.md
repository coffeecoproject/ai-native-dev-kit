# Storage Migration And Recovery Checklist

- [ ] Data ownership and resource scope are documented.
- [ ] Schema compatibility is reviewed.
- [ ] Migration, rollback, or recovery evidence exists.
- [ ] Backup or restore evidence exists where relevant.
- [ ] Deletion and retention behavior is documented.
- [ ] Data integrity checks exist for high-risk changes.
- [ ] Exceptions and residual risks are accepted by a human.
