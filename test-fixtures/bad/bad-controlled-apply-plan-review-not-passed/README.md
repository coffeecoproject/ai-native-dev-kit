# 1.41 Structured Evidence Schema Example

This example shows the apply evidence chain with machine-readable schema blocks:

1. `apply-plans/001-structured-workflow-assets.md`
2. `apply-readiness-reports/001-structured-workflow-assets.md`
3. `approval-records/001-structured-workflow-assets.md`

The readiness and approval records reference the Unified Apply Plan digest. The checkers validate the schema and the digest relationship while keeping the Markdown sections readable.
