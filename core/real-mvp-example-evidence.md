# Real MVP Example Evidence

Real MVP examples prove that the ordinary-user loop can produce a runnable first version, not only workflow records.

Each real MVP example must include:

- original one-sentence user goal;
- small human decisions;
- product brief;
- actual application files;
- local run instructions;
- verification command;
- ordinary first-slice card;
- product completeness report;
- final report;
- explicit non-production boundary.

Example evidence is local demo evidence only. It does not prove production readiness, real-user adoption, security, privacy, legal, payment, deployment, or release approval.
