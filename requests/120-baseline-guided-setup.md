# Request 120: Engineering and Environment Baseline Guided Setup

## Human Summary

Add a formal 1.2.0 upgrade that lets Codex recommend and prepare project-specific engineering and environment baselines while keeping humans responsible for decisions.

## Outcome Needed

- Baseline recommendation is read-only by default.
- Environment Baseline becomes first-class.
- Writes require reviewed plan.
- Task artifacts must prove baseline refs when touched.

## Non-goals

- Full source-code governance.
- Automatic environment discovery.
- Automatic CI/deploy edits.
- Secret handling or `.env` writes.
- Default BL2.
