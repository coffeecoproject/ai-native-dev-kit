# Frontend Review Checklist

- [ ] UI matches spec and product intent
- [ ] Uses existing design system/components
- [ ] Handles loading, empty, error, forbidden states
- [ ] Prevents duplicate submit
- [ ] Text fits containers on desktop and mobile
- [ ] Interactive controls have disabled/pending states
- [ ] Accessibility labels and keyboard flow are acceptable
- [ ] Responsive layout has been checked
- [ ] No unrelated visual redesign
