# Standard Baseline Selection Review

- [ ] Selected profiles are recorded.
- [ ] BL level is recorded.
- [ ] Standard packs and industrial overlays are separated.
- [ ] Standard pack IDs exist in `standard-baseline-packs/index.json`.
- [ ] Industrial overlay IDs exist in `industrial-packs/index.json`.
- [ ] Evidence gaps are recorded as `PENDING` when not proven.
- [ ] Human decision has exactly one status: `PENDING`, `APPROVED`, or `REJECTED`.
- [ ] The report does not authorize target-project writes.
- [ ] The report does not approve implementation.
- [ ] The report does not approve release or production.
- [ ] The report does not approve compliance, security, or privacy.
- [ ] Draft packs are not described as stable.
- [ ] `recommendedForBL` is not treated as default activation.
