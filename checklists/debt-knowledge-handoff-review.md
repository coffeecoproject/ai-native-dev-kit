# Debt & Knowledge Handoff Review Checklist

- [ ] Debt levels use only D0-D4.
- [ ] D3 blocks release review.
- [ ] D4 stops for human decision.
- [ ] Handoff explains what changed.
- [ ] Handoff explains why it changed.
- [ ] Handoff explains how to verify.
- [ ] Handoff explains where to continue next.
- [ ] Handoff explains what not to touch without approval.
- [ ] The report does not forgive debt.
- [ ] The report does not approve implementation.
- [ ] The report does not approve release or production.
- [ ] The report does not replace Review Loop or Safe Launch.
