# Verification Gate Checklist

General:

- [ ] lint
- [ ] typecheck
- [ ] unit tests
- [ ] integration tests
- [ ] build
- [ ] dependency/security checks if applicable

Scope:

- [ ] Only allowed files changed
- [ ] No forbidden modules changed
- [ ] No unrelated refactor
- [ ] No unapproved dependency

Spec:

- [ ] Acceptance criteria satisfied
- [ ] Non-goals respected
- [ ] API / interface contract matched
- [ ] Errors handled

UI / interface tasks:

- [ ] Loading state
- [ ] Empty state
- [ ] Error state
- [ ] Forbidden state
- [ ] Desktop layout checked
- [ ] Mobile layout checked
- [ ] No visible overlap
