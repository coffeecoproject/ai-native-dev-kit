# Scope Gate Checklist

- [ ] Task has one clear goal
- [ ] Related spec is linked
- [ ] Related eval is linked when required
- [ ] Allowed files/directories are listed
- [ ] Forbidden files/directories are listed
- [ ] Non-goals are explicit
- [ ] No unapproved dependency is required
- [ ] Stop conditions are listed
- [ ] Task level is assigned

Reject or split if any item is unclear.
