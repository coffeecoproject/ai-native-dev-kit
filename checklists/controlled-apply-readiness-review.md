# Controlled Apply Readiness Review Checklist

- [ ] The report references exactly one readable Unified Apply Plan.
- [ ] The report states `Can Codex apply now: No`.
- [ ] The report does not treat human confirmation as blanket approval.
- [ ] Low-risk candidate actions are separated from human-only or blocked actions.
- [ ] High-risk action types are never marked eligible for Codex-controlled apply.
- [ ] Backup, rollback, and verification readiness are present.
- [ ] Dirty worktree, missing target, missing evidence, missing rollback, or missing verification blocks readiness.
- [ ] Release, production, hooks, CI, migrations, secrets, payment, security, privacy, compliance, legal, and industrial packs remain human-only.
