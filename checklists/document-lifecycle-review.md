# Document Lifecycle Review Checklist

- [ ] The report states that it does not delete, move, archive, or change
  source of truth.
- [ ] Source-of-truth candidates cite evidence or request a human decision.
- [ ] Duplicate or stale documents are treated as candidates, not facts.
- [ ] Archive is the default suggestion before deletion.
- [ ] Deprecation suggestions are separate from deletion.
- [ ] What Not To Delete covers source-of-truth, agent rules, CI/hooks, release,
  legal, security, production, evidence, and sensitive-data documents.
- [ ] Any actual file move, archive, merge, deprecation note, or deletion is
  outside this report and requires a separate approved plan.
- [ ] The report does not claim documentation cleanup was completed.
