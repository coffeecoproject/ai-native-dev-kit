# Document Archive Apply Review Checklist

- [ ] Document Lifecycle evidence exists or the plan states no lifecycle evidence.
- [ ] Archive action status is `PLAN_ONLY` unless there is separate human approval.
- [ ] Source-of-truth replacement is named or marked pending.
- [ ] Link check plan exists before any apply claim.
- [ ] Archive Index preview exists.
- [ ] Rollback plan exists for every archive action.
- [ ] Protected docs are listed under What Not To Archive.
- [ ] Boundary says no deletion, no archive move now, no automatic link changes, and no cleanup approval.
- [ ] Outcome is `ARCHIVE_PLAN_RECORDED`, `NEEDS_HUMAN_DECISION`, or `BLOCKED`.
