# 1.88.2 Plan Review Consumer Integration

These examples show Execution Assurance, Completion Evidence, and Controlled Apply Readiness consuming the same Plan Review Gate result.

- `high-execution-assurance`: high-impact execution can claim done only after a passed Plan Review binding.
- `completion-evidence-plan-reviewed`: completion evidence carries the Plan Review binding alongside the BRC -> Verification -> Test -> Execution chain.
- `apply-readiness-plan-reviewed`: apply readiness requires a passed Plan Review binding before strict readiness review can pass.
