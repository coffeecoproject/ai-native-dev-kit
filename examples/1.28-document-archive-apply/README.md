# 1.28 Document Archive Apply Example

This example shows a controlled archive apply plan.

It does not move, delete, or archive files. It records what would need to be
checked before a future approved archive action:

- source evidence
- archive action plan
- link check plan
- archive index preview
- rollback plan
- human decisions

Run:

```bash
node scripts/check-document-archive-apply.mjs examples/1.28-document-archive-apply
```
