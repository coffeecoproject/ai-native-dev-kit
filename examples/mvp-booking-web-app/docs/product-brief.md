# Product Brief

## Goal

Build a simple local booking demo.

## Target User

- Visitor: submits a booking request.
- Operator: reviews submitted bookings.

## First Version Core Flow

1. Visitor enters name, phone, service, date, and time.
2. Visitor submits the booking.
3. Operator sees the booking in a local list.

## Out Of Scope

- Payment
- Login and roles
- SMS
- Production deployment
- Real customer data
