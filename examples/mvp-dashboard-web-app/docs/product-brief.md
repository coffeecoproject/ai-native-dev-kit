# Product Brief: dashboard web app

## Goal

Give a small team a local dashboard to understand work status before adopting real integrations.

## First Users

- Project owner
- Small internal team member

## Core Flow

1. Open the local dashboard.
2. Review active, blocked, and done counts.
3. Review current work items and owners.

## Out Of Scope

- Login and roles
- Live project management API integrations
- Production deployment
- Customer data
- Notification workflow
