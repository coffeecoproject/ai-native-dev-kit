# 1.67 Release Core Model Examples

These examples show Release Plan as a pure view layer.

- `web-preview`: new/light project release view
- `mini-program-review`: mini-program review view with platform submission kept human-owned
- `backend-api-handoff`: backend API handoff view with production deploy and migration kept human-owned
- `governed-existing-project-readonly`: existing governed project view with IntentOS Operating Mode active and project asset migration controlled
