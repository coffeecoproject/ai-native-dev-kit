# 1.80 Release Evidence Gate Examples

These examples show Release Evidence Gate as a review handoff layer.

- `web-preview-handoff`: enough evidence to hand a preview candidate to a release owner.
- `mini-program-review-handoff`: enough evidence to hand a mini-program review package to a release owner.
- `admin-production-review-blocked`: production-review target blocked because required release evidence is missing.

None of these examples approve release, deploy production, submit store review,
run migrations, record secrets, or prove real-user stability.
