# 1.70 Existing Project Governance Convergence Examples

These examples show how old projects can operate under IntentOS daily workflow
while keeping project-owned authority and stricter existing rules.

- `governed-web-admin`: a governed old web/admin project can converge daily
  workflow and baseline documents after review.
- `production-multiplatform`: production and platform release authority remains
  project-owned while IntentOS can provide the daily workflow.
- `dirty-worktree-blocked`: dirty work blocks convergence apply planning until
  the current changes are classified.
