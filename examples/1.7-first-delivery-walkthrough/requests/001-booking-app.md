# Request Card: Booking Mini App First Slice

## Human Summary

The user wants a simple booking mini app where customers can reserve a service time.

## Original User Message

I want to build a booking mini app for customers to reserve services.

## User Skill Level

Basic. The user should only need to choose, confirm, or decide.

## Goal

Create the first demo slice for booking:

- service list
- date and time selection
- customer name and phone input
- booking confirmation page

## Non-goals

- payment
- couponing
- staff scheduling
- production release
- customer announcement
- backend admin console

## Human Decisions Needed

| Decision | Options | Recommendation |
|---|---|---|
| First platform | Mini program / Web / Native app | Mini program first |
| Baseline level | BL0 / BL1 / BL2 | BL0 first |
| Payment | Include now / Defer | Defer |

## Acceptance Direction

The demo is successful if a customer can pick a service, choose a slot, enter contact details, and see a confirmation in local demo data.
