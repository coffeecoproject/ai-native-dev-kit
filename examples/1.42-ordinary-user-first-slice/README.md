# 1.42 Ordinary User First-Slice Example

This example shows the owner-facing first-slice path for a booking app idea.

It proves:

- the user sees a plain first-version recommendation;
- the card asks no more than three questions;
- backlog items are separated from first-version scope;
- verification is named before implementation;
- no write, implementation, release, CI, hook, BL2, or industrial-pack approval is granted.

Run:

```bash
node scripts/check-first-slice.mjs examples/1.42-ordinary-user-first-slice
```
