# Roadmaps

This directory stores roadmap and multi-phase planning documents.

Roadmaps describe intended direction. They are not release evidence and do not authorize implementation or project changes.

## Rule

Use roadmaps for direction. Use release records, current docs, and scripts for actual supported behavior.
