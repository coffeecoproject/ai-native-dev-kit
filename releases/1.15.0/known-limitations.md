# Release 1.15.0 Known Limitations

## Draft Status

All new platform standard packs are draft. Draft means they can be recommended, but they are not stable and do not prove real-project production validation.

## Scope

1.15.0 adds ordinary platform standard baseline packs. It does not add platform industrial packs, production certification, or real-project delivery evidence.

## Authority

Standard baseline selection does not approve:

- target-project writes
- implementation
- release or production
- security, privacy, legal, compliance, payment, tax, finance, HR, or migration decisions
- BL2 activation
- industrial overlays

## Existing Projects

Existing governed, production-sensitive, or dirty projects still require read-only mapping and gap review before any controlled apply path.

## Environment

`environment-standard` records facts and pending items. It must not store secret values, create or edit `.env`, invent deployment facts, or approve CI / CD changes.
