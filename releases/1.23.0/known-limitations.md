# 1.23.0 Known Limitations

- Hook Orchestration is plan-first; it does not install hooks.
- The resolver is heuristic and read-only.
- The checker validates recorded plans; it does not inspect every external automation system.
- H2/H3 installation remains a separate approved task.
- External API reviewer automation is not enabled.
- Blocking gates are not enabled.
- CI is not modified.
- Auto-fix is not enabled.
