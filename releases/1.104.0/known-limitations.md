# IntentOS 1.104.0 Known Limitations

- Runtime Trust proves execution provenance under recorded conditions, not semantic correctness of the test assertion.
- Automatic execution remains limited to the bounded local modes supported by 1.103.
- Docker, Kubernetes, serverless, provider, remote, release, and production lifecycle execution remains blocked.
- Historical consumer schemas remain readable but require regeneration before they can support a strict current completion claim.
- Missing technical runtime evidence is repaired or reported by Codex; it is not delegated to the zero-experience user.
