# 1.28.0 Known Limitations

- Document Archive Apply creates reviewable plans only; it does not move or delete files.
- Archive Index output is a preview unless a separate approved apply task writes it.
- Link checks are required before any future apply claim, but this phase does not automatically rewrite links.
- Source-of-truth decisions, archive approval, and rollback acceptance remain human-owned.
- Legal, security, production, release, CI, hook, and agent-governance docs stay protected by default.
