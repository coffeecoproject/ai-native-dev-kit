# 1.68.1 Known Limitations

- No npm package, installer, TUI, dashboard, hosted service, or GitHub Action is published by this release.
- Source-only adoption still requires cloning the repository and running local Node scripts.
- GitHub Release publication and private vulnerability reporting depend on repository-level GitHub operations outside source files.
- Dirty worktree protection cannot know the human owner of every local change; it only stops before higher-risk next actions.
- Plan-first recommendations do not approve applying a plan.
