# 1.68.0 Known Limitations

- The public entry is simpler, but the underlying governance system remains large.
- CLI command behavior is unchanged; this release does not introduce a command schema rewrite.
- The package remains repository-local and private; npm publishing is not part of 1.68.0.
- Security disclosure workflow, runtime sandboxing, and commercial licensing clarity remain future productization work.
- Advanced commands remain visible in CLI help because maintainers and CI still need exact command references.
