# Known Limitations: 1.0.0

## Human Summary

1.0.0 is a minimum productization release, not a claim that the kit has complete real-project proof.

## Limitations

- 10/10 real-project evidence has not been achieved.
- No migration apply command exists.
- `intentos migrate` is dry-run/write-plan only.
- Industrial packs remain `draft`.
- No industrial pack is promoted to `candidate` or `stable`.
- License explanation docs are not legal advice.
- External GPT/API reviewer automation is not implemented.
- Hook automation is not implemented.
- No package publishing is performed in this phase.
- Adoption evidence templates exist, but real adoption reports still need to be produced by actual projects.

## Safe Use

Use 1.0.0 for:

- controlled new project adoption
- read-only assessment of governed projects
- plan-first workflow asset updates
- first real project dogfood with evidence capture

Do not use 1.0.0 to claim:

- production readiness for every industrial pack
- legal approval for commercial usage
- automatic migration apply safety
- complete real-world validation

## Required Before 10/10 Evidence Release

- 2-3 real project adoption reports
- governed existing project read-only adoption report
- production-sensitive adapter trial
- industrial pack candidate promotion evidence
- public changelog backed by real adoption data
