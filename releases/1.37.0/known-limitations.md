# 1.37.0 Known Limitations

- Conversation-Native Ask defines Codex behavior, but it cannot force every external AI surface to follow the rule unless the surface reads these assets.
- The checker validates recorded Conversation Ask Cards; it does not inspect every live chat message automatically.
- The CLI `ask` command still exists for evidence and maintainers.
- This release does not add a write executor or automatic apply runner.
- This release does not prove real production project validation.
