# IntentOS 1.83.0 Known Limitations

- Task Governance is a classifier and router only.
- It does not authorize implementation, commit, push, release, production, CI,
  hooks, migrations, or target-project writes.
- It does not replace Business Rule Closure, Change Impact Coverage,
  Verification Plan, Test Evidence, Execution Assurance, Completion Evidence,
  Unified Closure, or User Delivery Console.
- It does not complete the later 1.83.1 source-binding expansion or 1.83.2
  enforcement phase.
- Project-native evidence mapping is supported, but correctness still depends
  on explicit refs, digests, task match, mapping state, and preserving stronger
  project rules.
