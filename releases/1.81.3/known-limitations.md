# IntentOS 1.81.3 Known Limitations

- `adopt` is still read-only. It gives a plain-language adoption result card,
  not migration approval.
- Plain-language Human Summary output does not replace machine-readable
  evidence, source-chain trace, or Outcome.
- Existing projects still need rule comparison, convergence evidence, adoption
  assurance, Unified Apply Plan, explicit approval, and readiness before any
  target-project workflow assets can change.
- 1.81.3 does not implement S1 docs-only writes, Adoption Continuation, or
  controlled native adoption review.
