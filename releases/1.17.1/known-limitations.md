# 1.17.1 Known Limitations

- This is a calibration patch, not a new capability layer.
- Baseline Decision Cards still do not approve target-project writes, implementation, release, production, BL2 activation, or high-risk domain decisions.
- `baseline-decision` prints only; saved records require an explicit `baseline-decision-card` artifact.
- CODEOWNERS ownership is declared, but enforcement depends on repository branch protection.
- Real project calibration is still needed before claiming strong recommendation accuracy.
