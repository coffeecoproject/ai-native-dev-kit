# IntentOS 1.79.0 Known Limitations

## Scope

User Delivery Console summarizes existing evidence. It does not create missing Business Rule Closure, Verification Plan, Test Evidence, Execution Assurance, Completion Evidence, Release Plan, or launch evidence.

## Current Limitations

- Status classification is conservative and source-signal based.
- The card can say what appears missing, but lower-level source systems remain authoritative.
- Launch review readiness is not release approval.
- Real-user stability still requires project-specific runtime, monitoring, and release evidence outside this card.
- The card is not a substitute for strict completion checks before Codex claims task completion.
