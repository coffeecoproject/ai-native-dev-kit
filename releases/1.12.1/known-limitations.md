# Known Limitations: 1.12.1

## Human Summary

1.12.1 closes metadata and guidance drift. It does not change the product scope or introduce the next standard baseline pack layer.

## Limitations

- CODEOWNERS still has no active owner rules because real GitHub handles require human governance decision.
- Change Boundary still checks recorded reports and optional git diff input; it is not an OS sandbox.
- Standard baseline packs remain a future design direction.
- Industrial packs remain governed by their own maturity metadata.
- This patch does not add production validation, automatic GPT/API review, automatic real-project scanning, release approval, risk acceptance, or commercial-use authorization.
