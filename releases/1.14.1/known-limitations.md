# Release 1.14.1 Known Limitations

## Scope

- 1.14.1 is a hardening patch, not a new baseline-pack phase.
- It does not add platform-specific complete baseline packs.
- It does not promote draft standard packs to candidate or stable.
- It does not make BL2 default.

## Evidence

- Evidence is limited to repository self-checks and fixtures.
- No real target project write, release, or production validation is claimed.
- Public URL allowlisting is intentionally narrow and can be expanded only with review.

## Governance Ownership

`.github/CODEOWNERS` intentionally contains no active owner rule yet. Maintainer handles must be confirmed before code ownership can become an enforced review gate.

## Authority

- Standard baseline selection remains a recommendation record.
- Human confirmation is still required before applying a selected baseline to a project.
- This patch does not approve target-project writes, implementation, release, production, compliance, security, privacy, or commercial use.
