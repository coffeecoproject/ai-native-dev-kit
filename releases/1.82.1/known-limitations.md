# IntentOS 1.82.1 Known Limitations

- Controlled Native Adoption Review remains review-only.
- Source trace digests are generated from resolver output and are not a
  substitute for future persisted apply-plan evidence.
- A selected deeper adoption recommendation still requires a separate Unified
  Apply Plan, Approval Record, Controlled Apply Readiness, and execution
  assurance chain before any write.
- This patch does not add S1/S2/S3/S4 apply behavior.
