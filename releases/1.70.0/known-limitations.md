# 1.70.0 Known Limitations

- Governance Convergence is a derived read-only view, not an apply mechanism.
- It depends on the quality of upstream Native Migration, Existing Rule Reconciliation, Workflow Next, and Release Plan evidence.
- It does not prove every old project rule was found.
- It does not resolve business, legal, security, privacy, compliance, tax, finance, HR, payment, permission, migration, data, provider, DNS, secret, or production ownership decisions.
- It does not automatically write an adoption anchor or AI log.
- It does not import historical evidence.
- It does not archive old documents or rewrite links.
- It does not make old projects identical to new projects; it defines a safe convergence path.
