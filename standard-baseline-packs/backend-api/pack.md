# Backend API Standard Pack

This draft standard pack defines ordinary backend API guardrails.

It applies when the project includes server APIs, domain DTOs, database schema, migrations, or API integration.

## Included Baselines

- `baselines/backend-api-contract-baseline.md`
- `baselines/backend-domain-data-baseline.md`

## Review Checklist

- `checklists/backend-api-review.md`

## Evidence Template

- `templates/backend-api-evidence.md`

## Boundary

- This pack can be recommended by AI.
- This pack is not active by default.
- Human selection is required before treating it as selected.
- This pack cannot authorize target-project writes.
- This pack cannot approve implementation.
- This pack cannot approve release or production.
- This pack cannot approve compliance, security, or privacy.
