# Release And Rollback Evidence Policy

Evidence should point to project release plans, verification commands, smoke results, rollback plans, or handoff records from the target project.

Pack files are not project evidence.

## Required For Confirmed Selection

- BL level evidence.
- Release or handoff scope evidence.
- Verification command evidence.
- Rollback or restore path evidence.
- Human decision boundary evidence.

If evidence is unavailable, record `PENDING` instead of inferring readiness.
