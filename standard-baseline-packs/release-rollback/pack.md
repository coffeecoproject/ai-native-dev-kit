# Release And Rollback Standard Pack

This draft standard pack defines ordinary release and rollback guardrails.

It applies when release, rollback, staging, production, handoff, or launch readiness is in scope.

## Included Baselines

- `baselines/release-readiness-baseline.md`
- `baselines/rollback-baseline.md`

## Review Checklist

- `checklists/release-rollback-review.md`

## Evidence Template

- `templates/release-rollback-evidence.md`

## Boundary

- This pack can be recommended by AI.
- This pack is not active by default.
- Human selection is required before treating it as selected.
- This pack cannot authorize target-project writes.
- This pack cannot approve implementation.
- This pack cannot approve release or production.
- This pack cannot approve compliance, security, or privacy.
