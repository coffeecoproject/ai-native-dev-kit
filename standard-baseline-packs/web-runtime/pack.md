# Web Runtime Standard Pack

This draft standard pack defines ordinary Web runtime guardrails.

It applies when a project includes Web UI, routing, state, API client behavior, forms, or runtime interaction.

## Included Baselines

- `baselines/web-runtime-baseline.md`
- `baselines/web-ui-state-baseline.md`
- `baselines/web-api-client-baseline.md`

## Review Checklist

- `checklists/web-runtime-review.md`

## Evidence Template

- `templates/web-runtime-evidence.md`

## Boundary

- This pack can be recommended by AI.
- This pack is not active by default.
- Human selection is required before treating it as selected.
- This pack cannot authorize target-project writes.
- This pack cannot approve implementation.
- This pack cannot approve release or production.
- This pack cannot approve compliance, security, or privacy.
